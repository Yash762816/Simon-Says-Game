# Simon Says Game

A browser-based implementation of the classic **Simon Says** memory game built with vanilla HTML, CSS, and JavaScript.

## How to Play
- Press **"S"** to start the game
- Watch the color sequence flash on screen
- Click the buttons in the **same order**
- Each level adds one more step to the sequence
- Wrong click? Game over!

## Features
- Increasing difficulty with each level
- Tracks and displays **Highest Score**
- Visual feedback for both game and user sequences
- Quit button to exit anytime

## Tech Stack
- HTML5
- CSS3
- JavaScript (DOM manipulation, Event Listeners)
